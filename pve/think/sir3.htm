Fractals

By Peter van Eijk, Nov 1997.

Sorry, Java applet here.This is a model of a 3-D H fractal, which is basically a 3-D rendition of a binary tree. I made one like this in welded steel, with 1 meter as its largest dimension.

The steel version of this is probably lying around the University of Twente somewhere. It makes a funny noise, because it appears to resonate on all frequencies.

The Java applet here makes it possible to rotate the model around, just click and drag.

This applet is one that can be found in the Java distribution 1.1.3 under demo/Wireframe.


Iterated function systems are a way to generate pictures like these. A description plus examples running under Java are in a separate page.
